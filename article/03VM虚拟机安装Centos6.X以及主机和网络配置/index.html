启动虚拟机

选择“Install or upgrade an existing system"，安装或升级已存在的系统。

是否对CD媒体进行测试，选择“Skip”跳过。

下一步。

选择语言，这里选择“English（English）”，英语。

选择键盘，默认，下一步。

选择安装设备，选择“Basic Storage Devices”，基本存储设备。

选择“Yes,Discard any data”，“是，忽略所有数据”，下一步。

设置主机名，这里默认，下一步。

选择时区，选择“Etc/GMT+8”，北京时区。

设置管理员root密码。

这里提示密码过于简单，选择“Use Anyway”，忽略。

选择安装类型，选择“Create Custom Layout”，创建自定义布局。

这里显示目前空闲控件为20473MB，点击”Create“创建分区。

选择“Standard Partition”，标准分区。

创建第一个挂载点：/boot，大小：128MB，文件系统类型：ext3。

继续创建分区，点击“Create”。

同样选择“Standard Partition”，标准分区。

创建swap分区，大小：2000MB（用于内存不够用时充当内存使用的）

继续创建/分区，点击“Create”。

同样选择“Standard Partition”，标准分区。

创建根目录“/”、大小：选中全部可用空间、文件类型：ext3。

分区后空间分布情况下图所示，下一步。

选择"Format"，格式化。

选择“Write changes to disk”,将更改写入到磁盘中。

下一步。

默然为“Minimal”，这里选择“Desktop”桌面，选中“Customize Now”，现在自定义。

选择需要的安装程序，下一步。

正在安装。

点击“Reboot”，重启系统。

正在重启CentOS系统。

启动后输入用户名和密码登陆系统。

虚拟机网络配置

宿主机的虚拟网关VMnet8的IP设置为自动获取。具体步骤为：右击点“属性”，然后选择IPV4那行，点击“属性"，更改成自动获取。

同理，修改“本地连接”也为自动获取。

“虚拟机”-“设置”，网络适配器选为NAT。

“编辑”-“虚拟网络编辑器”选中VMnet8，点选NAT（与虚拟机共享主机IP地址），把DHCP勾上（那两个框都勾上）。

点击“应用”。


开启宿主机（windows7）中的VMware DHCP Service 和VMware NAT Service服务。

net start "VMware DHCP Service"
net start “VMware NAT Service”

编辑 /etc/sysconfig/network

hostname直接命名：

编辑hosts文件，设置IP与主机名相对应

修改Linux的启动方式

当前是带图形化桌面的方式启动Linux，会占用不少的内存。

编辑 vi /etc/inittab 将id:5:initdefault:中的5改为3。

然后重启 reboot 电脑。

若在Shell 控制台想启动图形桌面，只要执行如下命令即可startx。
