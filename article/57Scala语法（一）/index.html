1、定义名称：
scala中只有两种方式定义名称：val，var

（1） val name:String="无名"
val修饰的是值，不可改变，String 类型可省略

（2） var money:Int=10000
var修饰的是变量，可重新赋值，Int类型可省略

money=100 #给money重新赋值

2.Scala里的数据类型是可以自动推导的

val 名称 = 值 只要=号后面有值，就不用写类型了，Scala中，它会自动从后面往前推导

3、常用数据类型以及转换：

Scala 与 Java有着相同的数据类型，下表列出了 Scala 支持的数据类型：

数据类型 描述
Byte 8位有符号补码整数。数值区间为 -128 到 127
Short 16位有符号补码整数。数值区间为 -32768 到 32767
Int 32位有符号补码整数。数值区间为 -2147483648 到 2147483647
Long 64位有符号补码整数。数值区间为 -9223372036854775808 到 9223372036854775807
Float 32 位, IEEE 754 标准的单精度浮点数
Double 64 位 IEEE 754 标准的双精度浮点数
Char 16位无符号Unicode字符, 区间值为 U+0000 到 U+FFFF
String 字符序列
Boolean true或false
Unit 表示无值，和其他语言中void等同。用作不返回任何结果的方法的结果类型。Unit只有一个实例值，写成()。
Null null 或空引用
Nothing Nothing类型在Scala的类层级的最低端；它是任何其他类型的子类型。
Any Any是所有其他类的超类
AnyRef AnyRef类是Scala里所有引用类(reference class)的基类

数据类型转换：
asInstanceOf 类型转换
isInstanceOf 判断类型
